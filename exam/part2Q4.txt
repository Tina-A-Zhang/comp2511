Factory Method pattern uses inheritance and relies on a subclass to handle the desired object instantiation, it is just a simple method used to creat objects in a class. Abstract Factory Method Patterns is a class delegates the responsibility of object instantiation to another object via composition. The main differece between a Factory Method and an Abstract Factory Method Patterns is that the factory methods is a single method, and an abstract factory method is an object. Factory Method is used to create one product only, however, Abstract Factory Method is about creating families of related or dependent products.
